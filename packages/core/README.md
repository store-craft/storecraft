This package has
- core types
- core API
- core database types
- core crypto types
- core storage types
- core mailer types
- core payment gateways types
- core taxes types
- core platform types
- core VQL types and logic
- core REST API controller

## Tests

While this package contains some tests,
Most tests to this package are in the `@storecraft/test-runner` package.
