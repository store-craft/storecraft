# Storage

`shelf` uses `storage` to 
* Save **images**
* Publish product **collections** 
* Publish **storefront** data

Out of the box, we support
* **Google** Storage
* **Cloudflare R2** Storage
* **Amazon S3** Storage
* Any general **S3 compatible** object storage

Copy the following with your own `UID`, then click **Publish**
```bash
rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /{allPaths=**} {
      allow read: if true;
      allow write: if request.auth.uid in ['YOUR_USER_UID']; 
    }
  }
}
```

## **Cloudflare R2** `Storage`

* Create an account with cloudflare.
* Create a bucket at R2 storage service
* Create an `access_key` and `secret` pair at cloudflare
* We also recommend connecting a subdomain to your bucket with cloudflare, (such as `https://cdn.your-domain.com`)

Next, at `shelf`, go to `settings` > `storage` card and enter the info
