# Well**come**

**Storecraft** is a next generation Commerce As Code Backend Platform.

You code your perfect commerce solution behaviour

Like this,

```ts
export const app = new App({
  general_store_name: "my-storecraft-app",
  auth_admins_emails: ["a@a.com"],
  auth_secret_access_token: process.env.auth_secret_access_token,
  auth_secret_refresh_token: process.env.auth_secret_refresh_token
})
.withPlatform(new NodePlatform())
.withDatabase(new SQLite())
.withStorage(new NodeLocalStorage())
.withMailer(new SendGrid({ apikey: process.env.SENDGRID_APIKEY }))
.withPaymentGateways(
  {
    'dummy_payments': new DummyPayments({ intent_on_checkout: 'AUTHORIZE' })
  }
)
.on(
  'orders/checkout/created',
  async (evt) => {

  }
)
```

## **Storecraft** is built to run on any Platform
- Serverless
- Serverfull
- Node.js, Deno, Bun
- AWS API Gateway
- Cloudflare Workers 
- Google Functions
- More will follow

## **Storecraft** can work with any Database 
- SQLite, MySQL, Postgres 
- MongoDB
- `Turso` Cloud sqlite database
- `Cloudflare D1` Cloud sqlite database
- `Neon` Cloud postgres database
- `Planetscale` Cloud mysql database
- You can roll your own integration.

## **Storecraft** can work with any Storage 
- Local FileSystem
- AWS S3
- Cloudflare R2 
- DigitalOcean Spaces
- MinIO
- Google Storage

## **Storecraft** can work with any Email provider,
- Http friendly
  - SendGrid
  - Resend
  - MailChimp
  - MailGun
- SMTP on Node.js / Bun / Deno
